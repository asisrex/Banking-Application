# Banking-Application
Functionalities
- Login
- Logout
- Forgot password
- Change Password
- Check balance
- Transfer Money
- Ministatement
- Search Transactions by Account number.
Used MVC ( Servlet , JSP ,HTML ) architecture to fullfill the requirement of the function.
